Términos y condiciones
1. RELACIÓN LABORAL
El repartidor opera como colaborador independiente. LavoBCN no establece relación laboral ni obligación de horarios fijos.

2. COMISIONES
LavoBCN no cobra comisión. Las propinas pertenecen íntegramente al repartidor.

3. GPS Y UBICACIÓN
El repartidor autoriza el uso del GPS durante el tiempo que esté activo. La ubicación se comparte con el admin y el cliente.

4. ZONA DE REPARTO
La zona asignada es Barcelona. Salir de zona de forma reiterada puede suspender el acceso.

5. DOCUMENTACIÓN
El repartidor debe mantener vigente su documentación. LavoBCN puede solicitar documentación adicional en cualquier momento.

6. COMPORTAMIENTO
El repartidor se compromete a tratar con respeto a clientes y lavanderías.

7. CÓDIGO DE ENTREGA
El repartidor debe verificar el código de 2 dígitos con el cliente antes de confirmar cada entrega.

8. DATOS PERSONALES
Los datos se usan exclusivamente para la gestión de la plataforma.

9. MODIFICACIONES
LavoBCN puede modificar estos términos con aviso previo de 15 días.

10. JURISDICCIÓN
Estos términos se rigen por la legislación española. Jurisdicción: Barcelona.
